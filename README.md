# SDEV2301 Student Lessons

This repository  **student-facing** lesson notes, examples, and practice activities for SDEV2301.

## How to use this repo
1. Open today’s lesson folder.
2. Read `README.md` for key concepts and a worked example.
3. Complete `PRACTICE.md`.
4. Use code in `examples/` as reference if you get stuck.

## Lessons
- Lesson 05A — Classes & Object Instantiation
- Lesson 05B — Constructors & Object State
- Lesson 07A — Properties, Fields & Encapsulation
