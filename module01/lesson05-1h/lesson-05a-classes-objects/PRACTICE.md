## Required Task: Book Class

### Requirements
Create a `Book` class with:
- Fields:
  - `Title` (string)
  - `Author` (string)
  - `Pages` (int)
- Method:
  - `PrintSummary()` prints a one-line summary of the book

In `Main`:
- Create **two Book objects**
- Assign values to their fields
- Call `PrintSummary()` on each

### Expected output (example)
Your output format can vary, but it should include title, author, and pages.
